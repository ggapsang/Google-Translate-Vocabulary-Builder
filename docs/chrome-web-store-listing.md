### 전용 목적 설명 (Single Purpose Description)

> 입력란: `전용 목적 설명` (0/1,000)

```
This extension helps users save and manage vocabulary words from Google Translate's dictionary panel.

When a user enters a word on translate.google.com and the dictionary panel appears, a "Save to Vocabulary" button is injected. Clicking it captures rich word data — definitions, examples, synonyms, part of speech, pronunciation, and translation context — and stores everything locally in the browser via chrome.storage.local.

Users manage their saved words through Chrome's side panel, which provides search, filtering (by language pair and part of speech), sorting, detail views, personal notes, and tags.

No data is sent to any external server. The extension operates exclusively on translate.google.com.
```

_(위 텍스트: 약 590자)_

### storage 사용 근거

> 입력란: `storage 사용 근거` (0/1,000)

```
The "storage" permission is required to persistently save and retrieve the user's vocabulary data using chrome.storage.local and chrome.storage.sync.

chrome.storage.local stores:
- Saved vocabulary words (word text, definitions, examples, synonyms, pronunciation, translation context)
- Vocabulary metadata (word count, last modified timestamp)
- Search index for fast word lookup
- Learning statistics

chrome.storage.sync stores:
- User preferences (UI language, theme, daily learning goals)

Without this permission, saved words would be lost when the browser closes. No data is transmitted to any external server — all operations use Chrome's built-in storage APIs only.
```

_(위 텍스트: 약 590자)_

---

### activeTab 사용 근거

> 입력란: `activeTab 사용 근거` (0/1,000)

```
The "activeTab" permission is used to interact with the currently active tab when the user clicks the extension's toolbar icon.

Specifically, it allows the extension to:
1. Open the side panel vocabulary manager in the context of the current tab.
2. Detect whether the user is on translate.google.com to enable or disable relevant features in the popup.

This permission follows the principle of least privilege — it only grants temporary access to the active tab upon explicit user action (clicking the extension icon), rather than requesting the broader "tabs" permission which would provide access to all open tabs.

The extension does not read, modify, or collect any data from the active tab through this permission. Tab interaction is limited to opening the side panel and checking the current URL.
```

### 호스트 권한 사용 근거

> 입력란: `호스트 권한 사용 근거` (0/1,000)
> 대상: `https://translate.google.com/*`

```
Host permission for "https://translate.google.com/*" is required because this extension's entire functionality operates exclusively on Google Translate.

The extension uses this permission to:
1. Run a content script on translate.google.com that observes the dictionary panel's DOM structure.
2. Parse word data from the dictionary panel — including definitions, example sentences, synonyms, part of speech, and pronunciation.
3. Inject a "Save to Vocabulary" button into the dictionary panel so users can save words with one click.
4. Read the source text and translated text from the translation interface to store as context alongside the saved word.

This permission is strictly limited to translate.google.com. The extension does not request access to any other website. It does not modify Google Translate's core functionality — it only reads dictionary data that is already displayed to the user and adds a save button.
```

#### `storage` 권한

**영어 (제출용):**
```
The "storage" permission is essential for saving and retrieving the user's vocabulary data. 

This extension saves word entries (word, definitions, examples, synonyms, user notes, tags) to chrome.storage.local so that the vocabulary persists across browser sessions. User preferences (theme, language, daily goals) are saved to chrome.storage.sync for cross-device consistency.

No data is sent to any external server. All storage operations are performed entirely within the browser's local storage APIs provided by the Chrome Extensions platform.
```

**한국어 (참고용):**
```
"storage" 권한은 사용자의 단어장 데이터를 저장하고 불러오기 위해 필수적으로 사용됩니다.

2. 사전 패널에서 단어 데이터(정의, 예문, 유의어, 품사)를 파싱합니다.
3. 구글 번역 페이지에 "Save to Vocabulary" 버튼을 주입합니다.
4. 번역 인터페이스에서 원문과 번역문을 읽어 컨텍스트로 저장합니다.

이 권한은 translate.google.com에만 한정되며, 다른 웹사이트에 대한 
접근 권한을 부여하지 않습니다. 확장 프로그램은 구글 번역의 정상 기능을 
수정하거나 방해하지 않으며, 사전 데이터를 읽고 저장 버튼을 추가하는 것이 전부입니다.
```

This extension has a single purpose: to help users save and manage vocabulary words from Google Translate's dictionary panel. It captures word data (definitions, examples, synonyms, part of speech) when users click the save button on translate.google.com, stores everything locally in the browser, and provides a side panel interface for searching, filtering, and reviewing saved words.
