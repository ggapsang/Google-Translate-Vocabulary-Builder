## Phase 2: 학습 기능

**목표**: 단어 테스트, 통계, Export 기능 추가

**기간**: 4주

### Stage 7-8: 테스트 모드

**개발자 작업**:
- [x] 테스트 설정 UI
  - 문제 수 선택 (예: 10, 20, 30문제)
  - 출제 범위 선택 (전체 단어, 최근 추가 단어, 자주 틀린 단어 등)
- [x] 주관식 빈칸 채우기 (Context Match) 테스트 로직 구현
  - 문제 출제용 데이터 가공 (`context.sourceText`와 `context.translatedText` 활용)
  - 사용자의 기본 언어(주 언어) 문장 표시 (예: "물론 대부분의 실내 공간에서는...")
  - 학습 언어 문장 표시 및 타겟 단어 마스킹 (예: "Of course, most indoor spaces will also fail to provide sufficient visibility of the sky for a GPS receiver to ____.")
  - 정답 검증 (대소문자 무시, 앞뒤 공백 제거 등 예외 처리)
  - 정답/오답에 대한 즉각적인 시각적 피드백 제공
  - 단어 메타데이터(`testResults`, `masteryLevel` 등) 업데이트 및 저장
- [x] 테스트 결과 및 복습 화면
  - 총점 표시
  - 오답(틀린 문제) 리스트 우선 표시 및 원래 예문/단어 확인
  - 틀린 단어 재학습/태그 추가 지원

**완료 기준**:
- 주관식 빈칸 채우기 10문제 테스트 전체 사이클 완료
- 주언어/학습언어 문장 렌더링 검증 완료
- 단어 입력 시 대소문자/스페이스 등 예외 처리 정상 작동
- 테스트 완료 후 해당 단어들의 Storage 메타데이터 업데이트 확인

**사용자 테스트 체크리스트**:

T1. 테스트 설정
- [x] 문제 수 선택 (10 / 20 / 30) 각각 정상 작동 확인
- [x] 출제 범위 선택 (전체 / 최근 추가 / 자주 틀린) 정상 작동 확인
- [x] 저장 단어 수 < 선택한 문제 수일 때 경고 또는 자동 조정 확인

T2. 문제 렌더링

- [x] 마스킹 위치가 원문 단어 위치와 일치하는지 확인
- [x] context 예문이 없는 단어의 처리 방식 확인 (에러 없이 넘어가는지)

- [x] 앞뒤 공백 포함 입력 (예: ` apple `) → 정답 처리 확인
- [x] 오답 입력 → 오답 피드백 + 정답 노출 확인

T4. 테스트 흐름

- [x] 마지막 문제 제출 후 결과 화면으로 자동 이동 확인
- [x] 전체 흐름 중 비정상 종료(화면 이탈 등) 없는지 확인

T5. 결과 화면
- [x] 총점(맞은 개수 또는 퍼센트) 올바르게 표시되는지 확인
- [x] 오답 목록이 정답 목록보다 우선 표시되는지 확인
- [x] 각 오답 항목에서 원래 예문 및 정답 확인 가능 여부
- [x] 틀린 단어 재학습 버튼 작동 확인
- [x] 틀린 단어 태그 추가 기능 작동 확인
- [x] "다시 테스트" 또는 "홈으로" 등 이후 액션 버튼 정상 작동

T6. Storage 메타데이터 업데이트
- [ ] 테스트 완료 후 각 단어의 `testResults` 기록 업데이트 확인
- [ ] 정답 시 `masteryLevel` 상승 / 오답 시 유지 또는 하락 확인
- [ ] 단어장에서 해당 단어의 숙련도 표시 변경 확인
- [ ] DevTools → Application → Storage에서 실제 데이터 직접 검증

T7. 엣지 케이스
- [ ] 저장된 단어 0개 상태에서 테스트 시작 시 처리 확인
- [x] 테스트 도중 Side Panel 닫았다가 다시 열었을 때 처리 확인
- [x] 같은 단어가 여러 언어 쌍으로 저장된 경우 중복 출제 여부 확인
- [x] 단어 수가 매우 적을 때 (예: 2~3개) 30문제 선택 시 처리 확인

사전 준비: `refs/sample-data.json`을 DevTools 콘솔에서 아래 코드로 주입

```js
fetch(chrome.runtime.getURL('refs/sample-data.json'))
   .then(r => r.json())
   .then(d => chrome.storage.local.set({
     vocabulary_words: d.vocabulary_words,
     vocabulary_stats: d.vocabulary_stats,
     vocabulary_meta: d.vocabulary_meta,
     vocabulary_index: d.vocabulary_index
   })).then(() => console.log('완료!'))
```
