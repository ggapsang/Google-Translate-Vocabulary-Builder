## 전체 개발 단계

```
Phase 1: MVP
  └─> Phase 2: 학습 기능
        └─> Phase 3: 고급 기능
              └─> Phase 4: 배포 & 운영 (지속)
```

## Phase 1: MVP (Minimum Viable Product)

**목표**: 기본적인 단어 저장 및 조회 기능 구현

**기간**: 4-6주

### Stage 1-2: 프로젝트 셋업 & 기본 구조

**개발자 작업**:
- [x] Chrome Extension 프로젝트 초기화
  - manifest.json 작성
  - 폴더 구조 생성
  - Git 저장소 설정
- [x] Content Script 기본 틀
  - 구글 번역 페이지 감지
  - DOM 접근 테스트
- [x] Storage API 래퍼 구현
  - chrome.storage.local 기본 CRUD
  - 에러 핸들링
- [x] Side Panel 기본 HTML/CSS
  - 레이아웃 구조
  - 기본 스타일링

**기술 자문 작업**:
- [x] 기획안 작성 완료
- [x] DOM 파싱 전략 상세화
- [x] 개발자 온보딩 문서 작성

**완료 기준**:
- 확장 프로그램이 Chrome에 로드됨
- 구글 번역 페이지에서 Content Script 실행 확인
- 데이터 저장/불러오기 테스트 성공

---

### Stage 3-4: 핵심 기능 구현

**개발자 작업**:
- [x] DOM 파싱 로직
  - 단어 추출
  - 품사 추출
  - 뜻 추출 (한국어, 영어)
  - 유의어 추출
  - 원문/번역문 추출
- [x] 저장 버튼 주입
  - 사전 패널에 버튼 추가
  - 클릭 이벤트 핸들러
  - 중복 저장 방지
- [x] 저장 로직 완성
  - 데이터 구조화
  - Storage에 저장
  - 저장 완료 피드백 (토스트)

**기술 자문 작업**:
- [x] DOM 셀렉터 검증
- [ ] 엣지 케이스 정리
  - 사전 정보 없는 경우
  - 여러 품사가 있는 경우
  - 네트워크 오류

**완료 기준**:
- 구글 번역에서 단어 클릭 → 사전 보기 → 저장 버튼 표시
- 저장 버튼 클릭 → 데이터 저장 → 토스트 표시
- 이미 저장된 단어는 "저장됨" 표시

---

### Stage 5-6: Side Panel 단어장

**개발자 작업**:
- [x] 단어 목록 렌더링
  - Storage에서 데이터 로드
  - 카드 형태로 표시
  - 스크롤 구현
- [x] 검색 기능
  - 단어/뜻으로 검색
  - 실시간 필터링 (Debounce)
- [x] 단어 상세 뷰
  - 모달 또는 전체 화면
  - 모든 정보 표시
  - 수정/삭제 기능
- [x] 기본 필터/정렬
  - 언어 쌍 필터
  - 최신순/가나다순 정렬

**기술 자문 작업**:
- [x] UX 플로우 검증
- [x] 성능 이슈 점검

**완료 기준**:
- Side Panel에서 저장된 단어 목록 확인
- 검색 기능 작동
- 단어 클릭 → 상세 정보 표시
- 단어 수정/삭제 가능

## Phase 2: 학습 기능

**개발자 작업**:

**기간**: 4주

**개발자 작업**:

  - easiness factor 계산
  - 다음 복습일 계산
- [ ] 복습 인터페이스
  - 오늘 복습할 단어 목록
  - "알고 있음" / "모르겠음" 버튼
  - 숙련도 업데이트
- [ ] 복습 통계
  - 복습률 추적
  - 복습 일정 캘린더

**참고 자료**:
- SuperMemo SM-2 Algorithm
- Anki 복습 시스템

**완료 기준**:
- 단어별 다음 복습일 계산
- 복습 완료 시 자동 업데이트
- 통계에 복습 데이터 반영

## Stage 성공 지표 (KPI)

### Phase 1 (MVP)
- 베타 테스터 5명 이상 모집
- 평균 저장 단어 수 50개 이상
- 심각한 버그 0개

### Phase 2 (학습 기능)
- 테스트 모드 이용률 30% 이상
- CSV Export 사용률 20% 이상

### Phase 3 (정식 출시)
- Chrome Web Store 평점 4.0+ / 5.0
- 주간 활성 사용자(WAU) 100명
- 평균 세션 시간 5분+
