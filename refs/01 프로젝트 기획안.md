# 구글 번역 학습 단어장 크롬 확장 프로그램 기획안
---
## 프로젝트 개요

### 프로젝트명
**Google Translate Vocabulary Builder** (가칭)

### 프로젝트 목적
구글 번역 사용 중 접하는 단어를 효과적으로 학습하고 관리할 수 있는 크롬 확장 프로그램 개발. 기존 구글 번역의 Phrasebook 기능을 넘어서는 학습 중심 단어장 솔루션 제공.

### 타겟 사용자
- 외국어 학습자 (영어, 일본어, 중국어 등)
- 번역 업무 종사자
- 해외 자료를 자주 읽는 개발자, 연구자, 학생
- 구글 번역을 일상적으로 사용하는 모든 사용자

### 핵심 가치 제안
1. **풍부한 정보 저장**: 단순 번역이 아닌 사전 정보(품사, 유의어, 영문 정의) 전체 저장
2. **완전한 문맥 보존**: 번역 요청한 원문 전체를 예문으로 저장
3. **학습 도구 제공**: 단어 테스트, 복습 알림 등 능동적 학습 지원
4. **데이터 소유권**: 로컬 저장 + CSV/Google Sheets Export로 완전한 데이터 통제권

## 기존 솔루션 분석

### 구글 번역 Phrasebook의 한계

| 기능        | 구글 Phrasebook   | 솔루션                    |
| --------- | --------------- | ---------------------- |
| 번역문 저장    | 가능              | 가능                     |
| 사전 정보 저장  | 불가능             | 가능 (품사, 유의어, 영문 정의)    |
| 원문 예문 저장  | 가능              | 가능 (전체 컨텍스트)           |
| 학습 테스트    | 불가능             | 가능 (객관식, 빈칸 채우기)       |
| 복습 알고리즘   | 불가능             | 가능 (Spaced Repetition) |
| Export 기능 | Google Sheets만  | 가능 (CSV + Sheets)      |
| 독립 UI     | 불가능 (번역 페이지 내만) | 가능 (사이드 패널, 탭, 팝업)     |
| 통계/분석     | 불가능             | 가능                     |

### 경쟁 우위

- 구글 번역과 완벽한 통합
- 별도 앱 설치 불필요
- 학습 맥락이 자동으로 보존됨
- 무료 + 오픈소스 지향
